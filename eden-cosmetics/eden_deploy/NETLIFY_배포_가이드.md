## 🌐 Netlify 배포 방법 (3분 완성!)

### 방법 1: 드래그 앤 드롭 (가장 쉬움! 추천!)

1. **Netlify 사이트 접속**
   - https://app.netlify.com/drop 로 이동

2. **로그인**
   - GitHub, GitLab, Bitbucket, 또는 이메일로 로그인
   - 무료 계정으로 충분합니다!

3. **폴더 드래그 앤 드롭**
   - `eden_deploy` 폴더 전체를 브라우저 창에 **드래그 앤 드롭**
   - 또는 "Browse to upload" 클릭 후 폴더 선택

4. **배포 완료!**
   - 자동으로 배포가 시작됩니다
   - 30초~1분 후 배포 완료
   - `https://랜덤이름.netlify.app` 형태의 URL이 생성됩니다

5. **사이트 이름 변경 (선택사항)**
   - Site settings → Change site name
   - 원하는 이름으로 변경 (예: `eden-mask.netlify.app`)

---

### 방법 2: Netlify CLI (명령어 방식)

PowerShell 실행 정책 문제로 현재 막혀있습니다.
대신 **방법 1 (드래그 앤 드롭)**을 사용하시면 더 빠르고 쉽습니다!

## 📱 모바일에서 확인하기

1. Netlify에서 제공하는 URL 복사

## 🔧 문제 해결

### Q: 이미지가 안 보여요!
A: 모든 이미지 파일이 `eden_deploy` 폴더에 있는지 확인하세요.

### Q: CSS가 적용 안 돼요!
A: `eden_styles.css` 파일이 `index.html`과 같은 폴더에 있는지 확인하세요.

### Q: 사이트 주소를 바꾸고 싶어요!
A: Netlify 대시보드 → Site settings → Change site name
